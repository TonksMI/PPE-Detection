KEY FINDINGS
------------

1. CNN PPENet dominates all metrics: 87.3% multi-class accuracy, macro AUC 0.9776, trained for
   100 epochs on 64x64 crops with AdamW + OneCycleLR.

2. Among ML models, the soft-voting Ensemble (SVM + RF + ExtraTrees + HistGBM) achieves the best
   multi-class accuracy at 79.5% and macro AUC 0.9497, confirming the value of model combination.

3. SVM achieves the best binary AUC (0.8915) and best binary accuracy (84.4%), making it the
   recommended ML-only baseline for the binary PPE present/absent task.

4. HistGBM leads ML models in binary AUC (0.9083) despite slightly lower accuracy than SVM,
   suggesting it produces better-calibrated probability scores for binary classification.

5. partial_ppe is the hardest class across all models (lowest AUC ~0.87-0.95), consistent with
   its visual ambiguity. helmet and safety_vest are the easiest (AUC ~0.95-0.99).

6. All models achieve AUC > 0.87 on every class, confirming strong discriminative power even
   for the weakest ML baselines. The CNN reaches AUC > 0.94 on all classes.

HOW THE REPORT WAS GENERATED
