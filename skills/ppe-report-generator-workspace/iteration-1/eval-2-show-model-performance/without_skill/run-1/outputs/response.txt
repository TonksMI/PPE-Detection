AUC SCORES (Binary PPE/no-PPE classification, from prod_roc_curves.png)

  SVM (PCA->RBF, binary):        AUC = 0.891  <-- BEST
  RandomForest (400 trees):      AUC = 0.876
  ExtraTrees (400 trees):        AUC = 0.875

The SVM has the highest AUC at 0.891. All three binary classifiers are competitive and well above random (0.5), with the SVM pulling clearly ahead especially in the low-FPR region (the top-left corner of the ROC plot), which is exactly where you want performance for a safety detection system — minimising false negatives at low false alarm rates.

Note: ROC/AUC was only computed for the binary models. The CNN and multi-class models do not have a single AUC value in the current results (multi-class AUC would require one-vs-rest curves per class, which were not generated).

CNN TRAINING (from prod_cnn_training.png and prod_cnn_confusion.png)

- Architecture: PPENet (64x64 input, 3 conv blocks 32->64->128 filters, AdaptiveAvgPool, FC 128->256->5)
- Training: 100 epochs, OneCycleLR scheduler (max_lr=1e-3, warmup 20%), label smoothing=0.05
- Best validation accuracy: 87.3%
- The training curve shows healthy convergence — train and val loss decrease together with no sign of severe overfitting. Val accuracy stabilises around epoch 60-70.
- Confusion matrix highlights: helmet (182/200 correct), no_ppe (188/200 correct), safety_vest (111/120 correct). Main confusion is between full_ppe and partial_ppe (20 misclassified), which is expected given their visual similarity.

SUMMARY — WHICH MODEL WINS?

Overall best model (multi-class accuracy + F1): CNN PPENet at 87.3% accuracy, macro F1=0.856
Best AUC (binary ppe/no-ppe discrimination): SVM (PCA->RBF) at AUC=0.891
Best ML-only model (no CNN): Ensemble (SVM+RF+ET+GBM) at 79.5% multi-class accuracy
Fastest model: ExtraTrees at 1.1-1.2s training time

The CNN is the clear winner for overall classification quality. The SVM is the best choice if you need a fast, interpretable binary detector with the highest AUC. The ensemble adds ~3pp over the best single ML model but takes ~200s to train.

Key area for improvement: partial_ppe and full_ppe remain the weak spots across all models (F1 in the 0.55-0.77 range). Class weighting or targeted augmentation on these two categories is the highest-leverage next step.
