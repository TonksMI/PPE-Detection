Key findings shown in the report:

CCTV Validation (6 unique scenes, from prod_cctv_results.csv):
  - Total people detected:    214
  - Total classified:         165
  - Violations flagged:        79
  - Violation rate:           47.9%
  - Note: scene 4.jpg is a large crowd (205 people, 156 classified, 75 violations)
    and is separated from the per-image bar chart to keep scale readable.

Model Performance (from prod_model_summary.csv):
  Multi-class (5 classes):
    - CNN PPENet (100ep):             87.33% accuracy, 0.856 Macro F1  [best]
    - Ensemble (SVM+RF+ET+GBM):       79.48% accuracy, 0.777 Macro F1
    - HistGBM (400 rounds):           76.72% accuracy, 0.745 Macro F1
    - SVM (PCA->RBF):                 76.31% accuracy, 0.745 Macro F1
    - RandomForest (400 trees):       73.00% accuracy, 0.708 Macro F1
    - ExtraTrees (400 trees):         71.76% accuracy, 0.693 Macro F1

  Binary (PPE vs No-PPE):
    - SVM (PCA->RBF):                 84.44% accuracy, 0.800 Macro F1  [best]
    - HistGBM (400 rounds):           83.20% accuracy, 0.790 Macro F1
    - RandomForest (400 trees):       81.40% accuracy, 0.719 Macro F1
    - ExtraTrees (400 trees):         78.37% accuracy, 0.622 Macro F1

The CNN PPENetFast is the clear winner on multi-class classification (87.33%), outperforming the best traditional ML model (Ensemble at 79.48%) by nearly 8 percentage points. For the simpler binary task, SVM achieves the best accuracy (84.44%) with a much faster training time (3 seconds vs 51.9 seconds for the CNN).
